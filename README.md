# Skyline
## 概要
Markdownファイルを元に記事を表示するブログです。Next.jsの新しいルーティング方式であるApp Routerの勉強のため制作しました。

お気に入りのビルについて自分なりにまとめてあります。記事を書くのが一番大変でした。

## 主な使用技術
- Next.js ver 13
- TypeScript
- Tailwind CSS
